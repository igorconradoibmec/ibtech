O computador só entende duas coisas

1

Verdadeiro

Ligado / Sim / Sinal alto

0

Falso

Desligado / Não / Sinal baixo

Bloco 2 — if / else

if e else: a decisão básica

int idade;
printf("Sua idade: ");
scanf("%d", &idade);

if (idade >= 18) {
  printf("Maior de idade\n");
} else {
  printf("Menor de idade\n");
}

Como funciona

O programa avalia a condição dentro dos parênteses.

Se for verdadeira (diferente de 0): executa o bloco do if.

Se for falsa (igual a 0): executa o bloco do else.

Condição entre (). Bloco entre {}. Instrução termina com ;

Bloco 2 — else if

Mais de duas opções: else if

float nota;
printf("Nota: ");
scanf("%f", &nota);

if (nota >= 9.0) {
  printf("A\n");
} else if (nota >= 7.0) {
  printf("B\n");
} else if (nota >= 5.0) {
  printf("C\n");
} else {
  printf("Reprovado\n");
}

Fluxo

O programa testa de cima pra baixo. No primeiro que for verdadeiro, entra e ignora o resto.

Se nota = 8.0:

  • nota >= 9.0? Não. Pula.
  • nota >= 7.0? Sim! Imprime "B".
  • Não testa mais nada.

O else final é o "senão genérico": pega tudo que não entrou em nenhuma condição.

Bloco 2 — Combinando condições

Usando && e || no if

Exemplo com E

if (idade >= 18 && tem_carteira) {
  printf("Pode dirigir\n");
}

As duas condições precisam ser verdadeiras.

Exemplo com OU

if (dia == 6 || dia == 7) {
  printf("Final de semana!\n");
}

Basta uma ser verdadeira.

Podem ser combinados: if (a > 0 && (b == 1 || c == 1)) — use parênteses pra controlar a ordem.

Bloco 2 — Exercício

Exercício: Calculadora simples

#include <stdio.h>

int main() {
  float a, b;
  char op;

  printf("Primeiro numero: "); scanf("%f", &a);
  printf("Operacao (+, -, *, /): "); scanf(" %c", &op);
  printf("Segundo numero: "); scanf("%f", &b);

  if (op == '+') {
    printf("Resultado: %.2f\n", a + b);
  } else if (op == '-') {
    printf("Resultado: %.2f\n", a - b);
  } else if (op == '*') {
    printf("Resultado: %.2f\n", a * b);
  } else if (op == '/' && b != 0) {
    printf("Resultado: %.2f\n", a / b);
  } else {
    printf("Operacao invalida\n");
  }
  return 0;
}

Bloco 3 — while

while: enquanto for verdadeiro, repete

int contador = 1;

while (contador <= 5) {
  printf("Contando: %d\n", contador);
  contador = contador + 1;
}
Saída:
Contando: 1
Contando: 2
Contando: 3
Contando: 4
Contando: 5

Como funciona

1. Testa a condição

2. Se verdadeira: executa o bloco

3. Volta pro teste

4. Repete até ser falsa

Se a variável de controle nunca mudar, o loop roda pra sempre (loop infinito). Sempre altere a variável dentro do bloco!

contador = contador + 1 pode ser escrito como contador++ (atalho do C).

Bloco 3 — for

for: quando você sabe quantas vezes

for (int i = 1; i <= 5; i++) {
  printf("Contando: %d\n", i);
}

Mesmo resultado do while anterior, mas em 3 linhas.

As 3 partes do for

int i = 1 → Inicialização (começa em 1)

i <= 5 → Condição (enquanto for verdadeira)

i++ → Incremento (soma 1 a cada volta)

i++ é o mesmo que i = i + 1. O ++ é atalho do C.

IbTech — Ibmec BH
Bloco 3 — Quando usar qual

while vs for

while

Quando você NÃO sabe quantas vezes vai repetir.

// Pedir senha até acertar
while (senha != 1234) {
  printf("Senha: ");
  scanf("%d", &senha);
}

for

Quando você SABE quantas vezes vai repetir.

// Tabuada do 5
for (int i = 1; i <= 10; i++) {
  printf("5 x %d = %d\n", i, 5 * i);
}

Bloco 3 — Exercício final

Exercício: Adivinhe o número

#include <stdio.h>

int main() {
  int secreto = 42;
  int chute = 0;
  int tentativas = 0;

  printf("Adivinhe o numero (1-100)!\n");

  while (chute != secreto) {
    printf("Seu chute: ");
    scanf("%d", &chute);
    tentativas++;

    if (chute > secreto) {
      printf("Menor!\n");
    } else if (chute < secreto) {
      printf("Maior!\n");
    }
  }
  printf("Acertou em %d tentativas!\n", tentativas);
  return 0;
}

O que esse programa usa

  • Variáveis (int)
  • printf e scanf
  • while (loop até acertar)
  • if / else if (dica maior/menor)
  • ++ (incremento)

Integra tudo que vocês aprenderam hoje numa coisa só.

Desafio extra

Limitar pra 10 tentativas. Se acabar, mostra "Você perdeu! O número era X".
